# Excom Frontend (MVP)

Stack: Next.js 14 + TypeScript + MUI

Scripts:
- dev: next dev -p 3000
- build: next build
- start: next start -p 3000

Pages (App Router):
- / (Home)
- /product/[id]
- /cart
- /checkout
- /seller/dashboard
- /ai (AI Concierge)

To run:
1. cd frontend
2. npm install
3. npm run dev
